**Creating the Third Space.**

Locall is a React Native mobile application designed to help users discover and create their "third space" ‚Äì the social surroundings separate from the two usual social environments of home ("first space") and the workplace ("second space").

## ‚ú® Features

- **Interactive Map**: Discover local events, pop-ups, and gatherings in real-time.
- **The Lounge**: A localized chat room for every event to break the ice before you arrive.

  - Auto-send icebreaker chips.
- **Event Creation**: Host your own gatherings with "Partiful-style" easy invites.

- **Authentication**: Seamless sign-in with Apple, Google, and Instagram (via Supabase).
